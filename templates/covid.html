COVID-19 Data Comparison


This page will allow chart based analysis of current, global COVID-19 data. Options are:

  • Zero Day Scatter plot which shows number of cases since the day of the first confirmed case in a given country.
  • Daily Cases bar chart showing incidence of new confirmed cases per day by country
  • Choropleth map, coloured based on deaths

Death per million population currently not working.
For all but Choropleth, where no input is required, simply enter the names of the countries you wish to analyse as comma separated values in the text box. Mis-spellings will cause errors.
